# 🏨 StayEase — Full-Stack Hotel Booking Platform

A full-stack hotel booking web application where travelers can discover, search, and book hotel rooms, while property owners can register their properties, manage room listings, and track reservations — all from a single platform.

## ✨ Features

### For Travelers
- 🔐 **User Authentication** — Secure sign-up, login, and profile management powered by Clerk
- 🔎 **Hotel Search** — Search by city, check-in/check-out dates, and number of guests directly from the homepage
- 🌟 **Hotel Discovery** — Explore featured hotel listings and browse a dedicated rooms page with filters for room type, price range, and sorting
- 🛏️ **Booking System** — View detailed room information, check availability, complete reservations, and track them in the **My Bookings** section
- 📧 **Booking Confirmation** — Receive confirmation emails after successful booking

### For Hotel Owners
- 🏨 **Hotel Registration** — Register properties through a dedicated popup form
- 📊 **Dashboard & Management** — Access a partner dashboard to manage hotel listings, add new rooms, and view booking details from the admin panel
